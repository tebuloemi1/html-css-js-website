My ambitions!

Ambition is a very important word to know about and understand. I define ambition as ones will or desire to accomplish or achieve a goal through a process that demands hard work, passion and determination.
My greatest ambition in this life is to become a best software engineer, whereby i will have a very extensive knowledge of programming languages, software development, and computer operating systems, and then apply engineering principles to software creation.

A good software engineer should be well-versed in:

  • Programming and coding
  • Fundamentals of computer science
  • Design and architecture
  • Algorithms and data structures
  • Information analysis
  • Testing software
  • Debugging software
